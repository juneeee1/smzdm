# 北京81销售体系内容分发系统 - 需求总览

## 项目目标
为销售人员提供专属推广物料的快速生成和分发系统，实现推广来源可追踪、物料可下载。

## 用户角色

### 个人（销售员）
- 注册字段：姓名、手机号、密码、所属门店、工号
- 登录后功能：
  - **门店信息区**：展示所属门店及个人信息
  - **渠道二维码申领**：展示专属二维码 + 个人渠道号，可下载
  - **营销内容申领**：展示推广海报（包含二维码、渠道号），可下载

### 公司
- 注册字段：公司名称、统一社会信用代码、联系人姓名、联系电话、密码、公司地址
- 登录后功能：
  - 展示公司渠道号（含基础信息：公司名称、注册时间）

## 渠道号生成规则
- 个人渠道号：`P` + 时间戳后6位 + 随机3位（例：P123456ABC）
- 公司渠道号：`C` + 时间戳后6位（例：C123456）

## 二维码与海报
- 二维码内容：`https://www.smzdm.com?channel={渠道号}`
- 海报：Canvas 合成（底图 + 二维码 + 渠道号文字 + 个人/门店信息）

## 技术约束
- Vue 3 + TypeScript + Vite
- Vue Router + Pinia
- qrcode 库生成二维码
- localStorage 本地存储，数据结构预留 id 字段

## 设计风格
- 简约高级
- 手机端为主，PC 端自适应
- 主色调以中性色为主，少量品牌色点缀
